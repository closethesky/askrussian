Копировать целиком содержимое одного из файлов в этой
директории (1.txt, 2.txt, 3.txt, 4.txt, 5.txt, 6.txt).

(О том, куда копировать, через несколько абзацев.)

Копировать содержимое только одного из файлов; многие
люди не начинают читать, увидев большой текст.

Иногда копировать сразу все пары миф+вопрос, они в файле
all.txt

Если текст не вставляется из-за ссылок, то файл
all_without_url.txt

[!!!] Все действия только в браузере "Tor"!

[!!!] Не вводить куда-либо личные данные (тел. номер,
адрес EMail, фамилию, пр.).

Браузер "Tor" бесплатно скачивается здесь:
https://torproject.org

Куда копировать

Искать сайты(!) по разным запросам, например,
Саратов установить кондиционер

* Создавать заказы в российских Интернет магазинах,
копировать содержимое файлов в комментарии к заказам.

* В формы обратной связи, часто в разделе "Контакты"
есть форма для отправки сообщений, часто возможно
отправить текст кнопкой "Заказать звонок", пр.

* В чаты на сайтах. Эти чаты иногда называют
"он-лайн консультант", "чат с оператором",
"чат с менеджером", пр.

* В сообщения EMail. Для отправки сообщений не(!)
российские сервисы с хорошей репутацией, например,
создать аккаунт в https://protonmail.com (выбирать
вариант "Free", для создания аккаунта не нужен
телефонный номер). Адреса EMail тоже искать
на сайтах, обычно в разделе "Контакты".
